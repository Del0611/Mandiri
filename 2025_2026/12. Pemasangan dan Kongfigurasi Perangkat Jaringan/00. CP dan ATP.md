# **Konfigurasi Perangkat Jaringan**

## **Capaian Pembelajaran (CP)**

Peserta didik mampu menerapkan konfigurasi perangkat jaringan dengan benar sesuai kebutuhan, meliputi pengaturan dasar, pengalamatan IP, serta pengujian konektivitas untuk memastikan jaringan dapat berfungsi secara optimal.

---

## **Tujuan Pembelajaran (TP)**

Peserta didik mampu melakukan konfigurasi perangkat jaringan yang mencakup pengaturan dasar perangkat, penentuan dan penerapan pengalamatan IP, serta melakukan pengujian konektivitas jaringan untuk memastikan sistem jaringan berjalan dengan baik dan optimal.

---

## **Alur Tujuan Pembelajaran (ATP)**

### **1. Pengaturan Dasar Perangkat Jaringan**

* Mengidentifikasi jenis dan fungsi perangkat jaringan (router, switch, access point).
* Menjelaskan konsep dasar konfigurasi perangkat jaringan.
* Melakukan konfigurasi dasar perangkat jaringan (hostname, password, interface).

Mengidentifikasi jenis dan fungsi perangkat jaringan seperti router, switch, dan access point, menjelaskan konsep dasar konfigurasi perangkat jaringan, serta melakukan konfigurasi dasar perangkat jaringan meliputi pengaturan hostname, password, dan interface.


### **2. Pengalamatan IP**

* Menjelaskan konsep alamat IP (IPv4).
* Mengidentifikasi kelas IP dan subnetting sederhana.
* Mengkonfigurasi IP address pada perangkat jaringan.
* Memverifikasi hasil konfigurasi IP.

Menjelaskan konsep alamat IP (IPv4), mengidentifikasi kelas IP dan melakukan subnetting sederhana, mengkonfigurasi IP address pada perangkat jaringan, serta memverifikasi hasil konfigurasi IP untuk memastikan pengalamatan jaringan berjalan dengan benar.


### **3. Pengujian Konektivitas Jaringan**

* Menjelaskan konsep konektivitas jaringan.
* Menggunakan tools pengujian jaringan (ping, traceroute).
* Menganalisis hasil pengujian konektivitas.
* Menyimpulkan kondisi jaringan berdasarkan hasil pengujian.

Menjelaskan konsep konektivitas jaringan, menggunakan tools pengujian jaringan seperti ping dan traceroute, menganalisis hasil pengujian konektivitas, serta menyimpulkan kondisi jaringan berdasarkan hasil pengujian tersebut.
