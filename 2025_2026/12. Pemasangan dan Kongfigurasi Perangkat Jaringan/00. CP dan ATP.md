# Konfigurasi Perangkat Jaringan

## Capaian Pembelajaran (CP)
Peserta didik mampu menerapkan konfigurasi perangkat jaringan dengan benar sesuai kebutuhan, meliputi pengaturan dasar, pengalamatan IP, serta pengujian konektivitas untuk memastikan jaringan dapat berfungsi secara optimal.

---

## Tujuan Pembelajaran (TP)

### 1. Pengenalan Konfigurasi Perangkat Jaringan
- Menjelaskan konsep dasar konfigurasi perangkat jaringan  
- Mengidentifikasi jenis-jenis perangkat jaringan (router, switch, access point)  
- Menjelaskan fungsi masing-masing perangkat jaringan  

### 2. Konfigurasi Dasar Perangkat Jaringan
- Mengakses perangkat jaringan (CLI/GUI)  
- Melakukan konfigurasi dasar (hostname, password)  
- Menyimpan konfigurasi perangkat  

### 3. Pengalamatan Jaringan
- Menjelaskan konsep IP Address  
- Mengkonfigurasi IP Address pada perangkat jaringan  
- Mengatur subnet mask dan gateway  

### 4. Pengujian dan Verifikasi
- Melakukan pengujian konektivitas jaringan (ping, traceroute)  
- Memverifikasi hasil konfigurasi perangkat  
- Mengidentifikasi dan memperbaiki kesalahan konfigurasi sederhana  

---

## Alur Tujuan Pembelajaran (ATP)

### Elemen 1: Pengenalan Perangkat dan Konfigurasi
**Alur TP:**  
Memahami konsep konfigurasi jaringan → Mengidentifikasi perangkat jaringan → Menjelaskan fungsi perangkat → Menentukan kebutuhan konfigurasi  

---

### Elemen 2: Konfigurasi Dasar
**Alur TP:**  
Mengakses perangkat jaringan → Melakukan konfigurasi dasar → Mengamankan perangkat (password) → Menyimpan konfigurasi  

---

### Elemen 3: Pengalamatan Jaringan
**Alur TP:**  
Memahami konsep IP Address → Menentukan alamat IP → Mengkonfigurasi IP pada perangkat → Mengatur gateway dan subnet  

---

### Elemen 4: Pengujian dan Verifikasi
**Alur TP:**  
Melakukan pengujian konektivitas → Memverifikasi konfigurasi → Mengidentifikasi kesalahan → Melakukan perbaikan → Menyimpulkan hasil konfigurasi
