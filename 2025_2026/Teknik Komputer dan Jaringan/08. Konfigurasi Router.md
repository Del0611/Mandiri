# Panduan Konfigurasi Dasar MikroTik (Basic Config)

Dokumen ini berisi langkah-langkah standar untuk melakukan konfigurasi awal pada RouterBoard MikroTik agar dapat berfungsi sebagai Gateway Internet.

## 1\. Persiapan dan Akses

Sebelum memulai, pastikan Anda memiliki aplikasi **Winbox** yang dapat diunduh dari situs resmi MikroTik.

1.  Hubungkan kabel internet (ISP) ke **Ether1**.
2.  Hubungkan kabel LAN dari komputer ke **Ether2** (atau port lain selain Ether1).
3.  Buka Winbox, klik tab **Neighbors**, lalu klik pada **MAC Address** router yang muncul.
4.  Login default:
      - User: `admin`
      - Password: (kosongkan)

> **Catatan:** Sangat disarankan untuk melakukan **Reset Configuration** (tanpa backup) jika router sebelumnya sudah pernah digunakan agar konfigurasi bersih.

-----

## 2\. Identitas Router (Identity)

Memberikan nama pada router agar mudah dikenali dalam jaringan.

  * **Menu:** `System` -\> `Identity`
  * **Perintah CLI:**
    ```bash
    /system identity set name=Router-Utama
    ```

-----

## 3\. Konfigurasi IP Address

Kita perlu menentukan alamat IP untuk jalur internet (WAN) dan jalur lokal (LAN).

### A. IP WAN (Static atau DHCP Client)

Jika ISP memberikan IP otomatis (DHCP):

  * **Menu:** `IP` -\> `DHCP Client` -\> Klik `+` -\> Interface: `ether1`
  * **Perintah CLI:**
    ```bash
    /ip dhcp-client add interface=ether1 disabled=no
    ```

### B. IP LAN (Lokal)

Menentukan alamat IP untuk jaringan internal.

  * **Menu:** `IP` -\> `Address` -\> Klik `+`
      - Address: `192.168.10.1/24`
      - Interface: `ether2`
  * **Perintah CLI:**
    ```bash
    /ip address add address=192.168.10.1/24 interface=ether2
    ```

-----

## 4\. Default Gateway (Routing)

*Langkah ini bisa dilewati jika Anda menggunakan DHCP Client pada langkah 3A.*
Jika IP WAN diset manual (Static), tambahkan gateway agar router tahu jalan menuju internet.

  * **Menu:** `IP` -\> `Routes` -\> Klik `+`
      - Dst. Address: `0.0.0.0/0`
      - Gateway: (Isi IP Gateway dari ISP, misal: `192.168.1.1`)
  * **Perintah CLI:**
    ```bash
    /ip route add gateway=192.168.1.1
    ```

-----

## 5\. Konfigurasi DNS

Agar router dapat menerjemahkan nama domain (seperti https://www.google.com/url?sa=E\&source=gmail\&q=google.com) menjadi alamat IP.

  * **Menu:** `IP` -\> `DNS`
      - Servers: `8.8.8.8`, `8.8.4.4`
      - Centang: `Allow Remote Requests`
  * **Perintah CLI:**
    ```bash
    /ip dns set servers=8.8.8.8,8.8.4.4 allow-remote-requests=yes
    ```

-----

## 6\. Firewall NAT (Masquerade)

Fungsi ini sangat penting agar perangkat di jaringan lokal (LAN) bisa mengakses internet menggunakan IP publik/WAN milik router.

  * **Menu:** `IP` -\> `Firewall` -\> Tab `NAT` -\> Klik `+`
      - Chain: `srcnat`
      - Out. Interface: `ether1` (Interface yang terhubung ke Internet)
      - Action: `masquerade`
  * **Perintah CLI:**
    ```bash
    /ip firewall nat add chain=srcnat out-interface=ether1 action=masquerade
    ```

-----

## 7\. DHCP Server (Opsional)

Agar komputer klien mendapatkan IP secara otomatis tanpa perlu setting manual.

  * **Menu:** `IP` -\> `DHCP Server` -\> Klik tombol `DHCP Setup`
    1.  DHCP Server Interface: `ether2`
    2.  DHCP Address Space: `192.168.10.0/24` (otomatis)
    3.  Gateway for DHCP Network: `192.168.10.1` (otomatis)
    4.  Addresses to Give Out: `192.168.10.2-192.168.10.254`
    5.  DNS Servers: `8.8.8.8`
    6.  Lease Time: `00:10:00`
  * **Perintah CLI:**
    ```bash
    /ip dhcp-server setup
    ```

-----

## 8\. Pengujian (Testing)

1.  **Cek Koneksi dari Router:**
    Buka `New Terminal` di Winbox, ketik:

    ```bash
    ping google.com
    ```

    Jika muncul balasan (reply), berarti router sudah terkoneksi ke internet.

2.  **Cek Koneksi dari Komputer Klien:**

      - Pastikan komputer diset "Obtain IP address automatically".
      - Buka CMD di Windows, ketik `ping google.com`.

-----

## 9\. Keamanan Dasar (Security)

Jangan biarkan router Anda tanpa pengamanan:

1.  **Ganti Password Admin:** `System` -\> `Password`.
2.  **Matikan Service Tak Terpakai:** `IP` -\> `Services` (Matikan telnet, ftp, www, api).

-----

**Selesai.** Router MikroTik Anda sekarang sudah siap digunakan sebagai gateway internet.
