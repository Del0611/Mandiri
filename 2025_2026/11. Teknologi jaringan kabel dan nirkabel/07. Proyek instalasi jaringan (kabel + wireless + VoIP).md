# 🏢 Modul Proyek: Instalasi Jaringan Terintegrasi (Kabel + Wireless + VoIP)

> **Skenario Proyek:** Sebuah perusahaan berskala kecil (atau sekolah) membutuhkan instalasi jaringan untuk ruangan kantor baru. Jaringan tersebut harus menyediakan koneksi PC via kabel, akses internet Wi-Fi untuk perangkat seluler, dan sistem telepon interkom antar-ruangan memanfaatkan jaringan IP yang ada.

---

## 🛠️ Bagian 1: Perencanaan & Kebutuhan Perangkat (Bill of Materials)

Sebelum masuk ke tahap instalasi fisik, berikut adalah komponen hardware dan software yang wajib disiapkan:

| Kategori | Nama Perangkat / Komponen | Fungsi Utama |
| --- | --- | --- |
| **Utama** | 1x Router Utama (Contoh: MikroTik RB951/Cisco 2811) | Gateway internet, DHCP Server, manajemen bandwidth. |
| **Utama** | 1x Switch Management (24 Port) | Distribusi kabel ke PC klien dan pembagi VLAN. |
| **Wireless** | 1x Access Point (AP) Indoor | Memancarkan sinyal Wi-Fi untuk area kantor/sekolah. |
| **VoIP** | 1x Server Mini / PC Server (Aplikasi FreePBX/3CX) | Central IP-PBX untuk mengatur ekstensi nomor telepon. |
| **VoIP** | 2x Softphone (Smartphone) & 1x IP Phone Fisik | Alat komunikasi/klien VoIP pengguna. |
| **Kabel** | Kabel UTP Cat6, Konektor RJ-45, Tang Crimping | Media transmisi data jaringan kabel fisik. |

---

## 📐 Bagian 2: Topologi Jaringan & Alokasi IP (Desain Sistem)

Untuk menghindari tabrakan data antara trafik data komputer biasa dengan data suara VoIP yang sensitif terhadap *delay*, jaringan akan dibagi menggunakan teknologi **VLAN (Virtual LAN)** pada satu Switch yang sama.

### Rencana Alokasi IP Address & VLAN:

* **VLAN 10 (Data - Kabel & Wi-Fi):** Network `192.168.10.0/24` (Untuk PC Kerja & HP)
* **VLAN 20 (Voice - VoIP):** Network `192.168.20.0/24` (Untuk Perangkat Telepon)

---

## 📝 Bagian 3: Langkah-Langkah Implementasi Proyek

Proyek ini diselesaikan melalui 4 tahapan kronologis terstruktur:

1. **1. Instalasi Kabel & Infrastruktur Fisik:** Physical Layer.
* Potong kabel UTP sesuai jarak antar-ruangan, lakukan *crimping* ujung kabel dengan konektor RJ-45 menggunakan standar **T568B (Straight)**.
* Tes kabel menggunakan LAN Tester untuk memastikan 8 pin menyala dengan benar.
* Hubungkan Router ke Switch, lalu tarik kabel dari Switch menuju PC client dan ke Access Point.


2. **2. Konfigurasi VLAN & DHCP di Router/Switch:** Switching & Routing.
* Masuk ke admin Switch, buat **VLAN 10** dan **VLAN 20**. Daftarkan port yang mengarah ke PC ke VLAN 10, dan port yang mengarah ke IP Phone ke VLAN 20.
* Di Router, buat interface *VLAN Trunking* dan aktifkan **DHCP Server** untuk masing-masing VLAN agar semua perangkat mendapatkan IP secara otomatis saat dicolokkan.


3. **3. Konfigurasi Jaringan Wireless (Wi-Fi):** WLAN Layer.
* Hubungkan laptop ke Access Point (AP). Masuk ke IP web admin AP tersebut.
* Set mode perangkat sebagai **Access Point Mode**.
* Ubah SSID menjadi `Kantor_Utama_WiFi`, pilih keamanan **WPA2-PSK**, dan masukkan password aman (misal: `SuksesBisnis2026`).


4. **4. Setup VoIP Server & Ekstensi Telepon:** Application Layer.
* Install software IP-PBX (misal: 3CX di Windows Server atau Asterisk).
* Buka web dashboard server VoIP tersebut, masuk ke menu **Extensions**, lalu buat 2 nomor ekstensi baru:
* Ekstensi `101` (Password: `user101`) -> Dialokasikan untuk Ruang Direktur.
* Ekstensi `102` (Password: `user102`) -> Dialokasikan untuk Ruang Staff.


* Install aplikasi **Zoiper** di smartphone staf dan login menggunakan data ekstensi tersebut dengan mengarahkan domain ke IP Server VoIP.


---

## 📊 Bagian 4: Pengujian & Komisioning (Testing & Quality Assurance)

Setelah semua tahap instalasi selesai, lakukan pengujian berikut untuk memastikan proyek layak diserahterimakan kepada klien:

1. **Uji Jaringan Kabel:** Pastikan PC Klien mendapatkan IP dari subnet `192.168.10.X`, lakukan `ping google.com` untuk memastikan internet aktif.
2. **Uji Jaringan Wireless:** Sambungkan smartphone ke `Kantor_Utama_WiFi`, pastikan proses autentikasi berhasil dan dapat berselancar di internet dengan lancar.
3. **Uji VoIP (Paling Krusial):**
* Buka aplikasi Zoiper di smartphone Ruang Direktur (`101`) dan Ruang Staff (`102`). Pastikan indikator akun berwarna hijau (*Registered*).
* Lakukan panggilan dari `101` ke `102`. Pastikan telepon berdering, suara terdengar jelas di kedua sisi (*Two-Way Audio*), tanpa ada gejala putus-putus (*jitter*).



---

> 💡 **Rekomendasi Maintenance:** > Agar layanan VoIP tetap jernih saat seluruh karyawan sedang mengunduh file besar, aktifkan fitur **QoS (Quality of Service)** pada Router Utama. Berikan prioritas bandwidth tertinggi bagi paket data yang berasal dari **VLAN 20 (Voice)**.
