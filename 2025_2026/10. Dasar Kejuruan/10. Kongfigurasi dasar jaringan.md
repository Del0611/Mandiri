# 🌐 Materi Praktikum: Konfigurasi Dasar Jaringan (IP Address dan Konektivitas)

## 🎯 Tujuan Praktikum

Setelah menyelesaikan praktikum ini, peserta didik mampu:

* 📌 Memahami konsep dasar alamat IP (IP Address).
* 📌 Melakukan konfigurasi IP Address secara manual pada komputer.
* 📌 Memverifikasi konfigurasi jaringan menggunakan perintah dasar.
* 📌 Menguji konektivitas antar perangkat dalam jaringan menggunakan perintah *ping*.

---

## 🛠️ Alat dan Bahan

* 💻 Laptop atau PC
* 🌐 Kabel UTP
* 🔀 Switch atau Hub
* 🖥️ Minimal 2 komputer yang terhubung dalam satu jaringan
* ⚙️ Sistem Operasi Windows/Linux

---

## 📚 Dasar Teori

### 🌍 IP Address

IP Address (*Internet Protocol Address*) adalah alamat logis yang digunakan untuk mengidentifikasi perangkat dalam jaringan komputer.

Contoh IP Address:

```text
192.168.1.10
192.168.1.20
```

### 🧩 Komponen IP Address

| Komponen    | Contoh        |
| ----------- | ------------- |
| IP Address  | 192.168.1.10  |
| Subnet Mask | 255.255.255.0 |
| Gateway     | 192.168.1.1   |
| DNS Server  | 8.8.8.8       |

### 📡 Konektivitas Jaringan

Konektivitas jaringan adalah kemampuan dua atau lebih perangkat untuk saling berkomunikasi dan bertukar data dalam suatu jaringan.

---

# 🔧 D. Langkah Kerja Praktikum

## 🔹 Langkah 1: Persiapan Perangkat

1. 🔌 Hubungkan kedua komputer ke switch menggunakan kabel UTP.
2. 💡 Pastikan lampu indikator pada port switch menyala.
3. 🖥️ Nyalakan seluruh perangkat yang akan digunakan.

---

## 🔹 Langkah 2: Konfigurasi IP Address

1. ⚙️ Buka menu **Network Settings** pada komputer.
2. 🌐 Pilih adaptor jaringan yang digunakan.
3. ✏️ Atur IP Address secara manual dengan konfigurasi berikut:

### Komputer 1

| Parameter   | Nilai          |
| ----------- | -------------- |
| IP Address  | 192.168.10.1   |
| Subnet Mask | 255.255.255.0  |
| Gateway     | 192.168.10.254 |

### Komputer 2

| Parameter   | Nilai          |
| ----------- | -------------- |
| IP Address  | 192.168.10.2   |
| Subnet Mask | 255.255.255.0  |
| Gateway     | 192.168.10.254 |

4. 💾 Simpan konfigurasi yang telah dibuat.

---

## 🔹 Langkah 3: Verifikasi Konfigurasi IP

1. ⌨️ Buka **Command Prompt** atau **Terminal**.
2. 📋 Jalankan perintah berikut:

```bash
ipconfig
```

atau pada Linux:

```bash
ifconfig
```

atau

```bash
ip addr
```

3. ✅ Pastikan alamat IP yang tampil sesuai dengan konfigurasi yang telah dibuat.

---

## 🔹 Langkah 4: Pengujian Konektivitas Jaringan

1. 🖥️ Pada Komputer 1, buka Command Prompt.
2. 📡 Jalankan perintah:

```bash
ping 192.168.10.2
```

3. 🖥️ Pada Komputer 2, jalankan perintah:

```bash
ping 192.168.10.1
```

4. ✅ Perhatikan hasil pengujian.

Contoh hasil berhasil:

```text
Reply from 192.168.10.2: bytes=32 time<1ms TTL=128
Reply from 192.168.10.2: bytes=32 time<1ms TTL=128
Reply from 192.168.10.2: bytes=32 time<1ms TTL=128
Reply from 192.168.10.2: bytes=32 time<1ms TTL=128
```

---

## 🔹 Langkah 5: Analisis Hasil

### ✅ Jika Berhasil

* Kedua komputer dapat saling berkomunikasi.
* Konfigurasi IP Address sudah benar.
* Koneksi fisik jaringan berfungsi dengan baik.

### ❌ Jika Gagal

Periksa kembali:

* 🔌 Kabel jaringan terpasang dengan benar.
* 🌐 IP Address berada pada subnet yang sama.
* ⚙️ Network Adapter aktif.
* 🔥 Firewall tidak memblokir koneksi ping.
