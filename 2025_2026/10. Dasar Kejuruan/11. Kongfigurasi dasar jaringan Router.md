# 🌐 Materi Praktikum: Konfigurasi Dasar Router

## 🎯 Tujuan Praktikum

Setelah menyelesaikan praktikum ini, peserta didik mampu:

- 📌 Memahami fungsi router dalam jaringan komputer.
- 📌 Melakukan konfigurasi IP Address pada router.
- 📌 Mengaktifkan layanan DHCP Server.
- 📌 Menghubungkan perangkat klien ke router.
- 📌 Menguji konektivitas jaringan menggunakan perintah *ping*.

---

## 🛠️ Alat dan Bahan

- 🌐 Router (MikroTik/Cisco/TP-Link)
- 💻 Laptop atau PC
- 🔌 Kabel UTP
- ⚡ Adaptor daya router
- 🔀 Switch (Opsional)
- 🖥️ Perangkat klien

---

## 📚 Dasar Teori

### 🌍 Pengertian Router

Router adalah perangkat jaringan yang berfungsi menghubungkan dua atau lebih jaringan yang berbeda serta mengatur lalu lintas data agar sampai ke tujuan yang tepat.

### 🎯 Fungsi Router

- 📡 Menghubungkan jaringan lokal dengan internet.
- 🔀 Mengarahkan paket data antar jaringan.
- 🌐 Memberikan alamat IP secara otomatis melalui DHCP.
- 🔒 Mengatur keamanan dan akses jaringan.

### 🧩 Parameter Dasar Router

| Parameter | Contoh |
|-----------|---------|
| IP Address | 192.168.10.1 |
| Subnet Mask | 255.255.255.0 |
| Gateway | 192.168.10.1 |
| DNS Server | 8.8.8.8 |

# 🔧 D. Langkah Kerja Praktikum

## 🔹 Langkah 1: Persiapan Perangkat

1. 🔌 Hubungkan adaptor daya ke router.
2. 💡 Pastikan lampu indikator Power menyala normal.
3. 🌐 Hubungkan laptop ke port LAN router menggunakan kabel UTP.
4. 🖥️ Nyalakan laptop dan router.

---

## 🔹 Langkah 2: Mengakses Router

1. 🌍 Buka aplikasi browser atau WinBox (untuk MikroTik).
2. ✏️ Masukkan alamat IP default router.

Contoh:

```text
192.168.88.1
```

3. 🔑 Login menggunakan akun administrator.

Contoh:

```text
Username : admin
Password : (kosong)
```

---

## 🔹 Langkah 3: Konfigurasi IP Address Router

1. ⚙️ Masuk ke menu **IP → Addresses**.
2. ➕ Klik tombol **Add (+)**.
3. Masukkan konfigurasi berikut:

| Parameter | Nilai |
|-----------|---------|
| IP Address | 192.168.10.1/24 |
| Interface | ether2 |

4. 💾 Klik **Apply** dan **OK**.

---

## 🔹 Langkah 4: Konfigurasi DHCP Server

1. 📡 Buka menu **IP → DHCP Server**.
2. Klik **DHCP Setup**.
3. Pilih interface yang digunakan untuk jaringan lokal.
4. Tentukan rentang alamat IP.

Contoh:

| Parameter | Nilai |
|-----------|---------|
| Network | 192.168.10.0/24 |
| Gateway | 192.168.10.1 |
| IP Pool | 192.168.10.10 - 192.168.10.100 |
| DNS Server | 8.8.8.8 |

5. 💾 Selesaikan konfigurasi hingga DHCP Server aktif.

---

## 🔹 Langkah 5: Verifikasi Konfigurasi

1. ⌨️ Buka Command Prompt pada komputer klien.
2. Jalankan perintah:

```bash
ipconfig
```

3. ✅ Pastikan komputer memperoleh IP Address secara otomatis dari router.

Contoh:

```text
IPv4 Address : 192.168.10.10
Default Gateway : 192.168.10.1
```

---

## 🔹 Langkah 6: Pengujian Konektivitas

1. ⌨️ Jalankan perintah berikut:

```bash
ping 192.168.10.1
```

2. Perhatikan hasil pengujian.

Contoh hasil berhasil:

```text
Reply from 192.168.10.1: bytes=32 time<1ms TTL=64
Reply from 192.168.10.1: bytes=32 time<1ms TTL=64
Reply from 192.168.10.1: bytes=32 time<1ms TTL=64
Reply from 192.168.10.1: bytes=32 time<1ms TTL=64
```

---

## 🔹 Langkah 7: Analisis Hasil

### ✅ Jika Berhasil

- Router dapat diakses dari komputer klien.
- DHCP Server berjalan dengan baik.
- Komputer mendapatkan IP Address secara otomatis.
- Koneksi jaringan berfungsi normal.

### ❌ Jika Gagal

Periksa kembali:

- 🔌 Kabel jaringan terpasang dengan benar.
- 🌐 Konfigurasi IP Address router sesuai.
- 📡 DHCP Server aktif.
- ⚙️ Interface jaringan aktif.
- 🔥 Firewall tidak memblokir koneksi.
