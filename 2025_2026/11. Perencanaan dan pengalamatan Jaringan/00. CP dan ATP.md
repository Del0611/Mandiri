---

# 🌐 2. Topologi Jaringan di Cisco

## 📖 Jenis Topologi

### ⭐ Star Topology
- Menggunakan switch sebagai pusat

### 🚌 Bus Topology
- Semua perangkat terhubung dalam satu jalur

### 🔄 Ring Topology
- Perangkat membentuk lingkaran

### 🕸️ Mesh Topology
- Setiap perangkat saling terhubung

## 🛠️ Praktik di Cisco:
- Menambahkan PC, switch, router
- Menghubungkan perangkat dengan kabel
- Menguji koneksi (ping)

---

# 🏗️ 3. Arsitektur Jaringan

## 🖥️ Client-Server
- Server menyediakan layanan
- Client menggunakan layanan

### 🛠️ Praktik Cisco:
- Menambahkan server (DHCP, DNS)
- Menghubungkan client ke server
- Menguji koneksi jaringan

---

## 🤝 Peer-to-Peer
- Komputer saling terhubung langsung

### 🛠️ Praktik Cisco:
- Menghubungkan 2 PC langsung
- Mengatur IP manual
- Menguji komunikasi

---

# 🌍 4. IP Address di Cisco

## 📖 Pengertian
IP Address digunakan untuk memberi alamat pada setiap perangkat jaringan.

## 🛠️ Praktik:
- Konfigurasi IP pada PC
- Pengaturan subnet mask
- Pengaturan gateway

### 📌 Contoh:
```text
IP Address : 192.168.1.10  
Subnet Mask: 255.255.255.0  
Gateway    : 192.168.1.1  
````

---

# 🔢 5. Subnetting di Cisco

## 📖 Pengertian

Subnetting adalah membagi jaringan besar menjadi beberapa jaringan kecil.

## 🛠️ Praktik:

* Membagi network di router
* Mengatur IP sesuai subnet
* Menguji koneksi antar subnet

---

# 🌐 6. CIDR di Cisco

## 📖 Pengertian

CIDR adalah metode penulisan IP dengan prefix.

### 📌 Contoh:

```text
192.168.1.0/24
```

## 🛠️ Praktik:

* Mengatur jaringan berdasarkan prefix
* Menguji jumlah host yang tersedia

---

# 🔧 7. VLSM di Cisco

## 📖 Pengertian

VLSM digunakan untuk membagi IP sesuai kebutuhan masing-masing jaringan.

## 🛠️ Studi Kasus:

* Lab 1: 50 host
* Lab 2: 25 host
* Kantor: 10 host

## 🛠️ Praktik:

* Pembagian subnet berbeda
* Konfigurasi router
* Pengujian koneksi antar jaringan
